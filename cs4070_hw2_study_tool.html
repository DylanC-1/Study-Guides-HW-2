Min-Max Normalization

Scales all values proportionally into the range [new_min, new_max] — here [0, 1].

The Formula Q1a

Formula
v' = (v − min_A) / (max_A − min_A) × (new_max − new_min) + new_min

Since new_min = 0 and new_max = 1, the formula simplifies to:

v' = (v − min_A) / (max_A − min_A)
  1. Find min_A (smallest value) and max_A (largest value) in the dataset
  2. Compute the range: max_A − min_A
  3. For each value v, subtract min_A then divide by the range

Worked Solution — Data: 200, 300, 400, 600, 1000

max_A = 1000
range = 1000 − 200 = 800

Z-Score Normalization

Centers data around the mean with standard deviation as the unit. Tells you how many standard deviations each value is from the mean.

The Formula Q1b

Formula
v' = (v − μ) / σ

where μ = mean, σ = standard deviation
  1. Calculate the mean (μ): sum all values, divide by count (n)
  2. For each value, find the squared deviation: (v − μ)²
  3. Calculate variance: average of all squared deviations → Σ(v−μ)² / n
  4. Take the square root of variance to get σ
  5. Apply the formula: subtract mean, divide by σ

Worked Solution — Data: 200, 300, 400, 600, 1000

STEP 1 — MEAN (μ)
Sum = 200 + 300 + 400 + 600 + 1000 = 2500
n = 5
μ = 2500 / 5 = 500
STEP 2 — STANDARD DEVIATION (σ)
vv − μ(v − μ)²
200200 − 500 = −30090,000
300300 − 500 = −20040,000
400400 − 500 = −10010,000
600600 − 500 = 10010,000
10001000 − 500 = 500250,000
Σ(v−μ)² = 90,000 + 40,000 + 10,000 + 10,000 + 250,000 = 400,000
Variance = 400,000 / 5 = 80,000
σ = √80,000 ≈ 282.84
STEP 3 — APPLY Z-SCORE
v(v − 500) / 282.84v' (z-score)
200−300 / 282.84−1.061
300−200 / 282.84−0.707
400−100 / 282.84−0.354

1000500 / 282.84+1.768

Normalization by Decimal Scaling

Moves the decimal point by dividing by a power of 10 (10^j) large enough to make all values fall in [−1, 1].

The Formula Q1c

Formula
v' = v / 10^j

where j = smallest integer such that max(|v'|) < 1
  1. Find the largest absolute value in the dataset
  2. Find j: the number of digits in that largest value (i.e., how many times you divide by 10)
  3. Divide every value by 10j
Key rule: j must be large enough so that all normalized values have absolute value < 1. The max value in our data is 1000, which has 4 digits, so j = 4, giving 10⁴ = 10,000.

Worked Solution — Data: 200, 300, 400, 600, 1000

FINDING j
max|v| = 1000
Try j = 3: 10³ = 1000 → 1000/1000 = 1.0 ✗ (not < 1)
Try j = 4: 10⁴ = 10,000 → 1000/10,000 = 0.1 ✓ (< 1)
j = 4
v (original)v / 10⁴ (÷ 10,000)v' (normalized)
200200 / 10,0000.02
300300 / 10,0000.03
400400 / 10,0000.04
600600 / 10,0000.06
10001,000 / 10,0000.10
✓ Sanity check: All values are between 0 and 1 (the max is 0.10 which is < 1). ✓
